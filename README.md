# CareerIQ — AI Career Transition Co-Pilot
Built on Microsoft Foundry · gpt-4.1-mini · Real-time streaming

## Setup
1. Copy `.env.example` to `.env` and fill in your Foundry endpoint
2. Install: `pip install -r requirements.txt`
3. Login: `az login --use-device-code`
4. Run: `python app.py`
5. Open: `http://localhost:5000`

## Features
- Real-time streaming responses
- Multi-turn conversation memory
- Skill gap analysis
- 90-day Microsoft Learn plans
- Cover letter drafting
- Market insights
- Professional dark UI

## Built with
- Microsoft Foundry (gpt-4.1-mini)
- Azure AI Projects SDK
- Flask + Server-Sent Events
- Vanilla JS (no framework needed)
